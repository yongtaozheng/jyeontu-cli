## 测试5：混合内容

这是一个包含数学公式的段落：$f(x) = x^2 + 2x + 1$，它的导数是 $f'(x) = 2x + 2$。

下面是代码示例：

```python
# Python 代码示例
def hello_world():
    print("Hello, World!")
    return "Hello, World!"

# 这是一个注释
result = hello_world()
print(result)
```

下面是数学公式：

$$
f(x) = x^2 + 2x + 1 = (x + 1)^2
$$
